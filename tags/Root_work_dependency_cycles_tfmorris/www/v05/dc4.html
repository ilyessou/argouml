Step Three: Download Argo/UML v0.5.2 Bytecodes

Download the main jar file:

  • argouml.zip (0.6 Mb) You must unzip this file to get argouml.jar.

Please keep in mind that this is an early release of a research project, not a commercial CASE tool. Argo/UML has many interesting and useful features, but it lacks some of the standard features commonly found in current CASE tools.


Step Four: Download Optional jar Files

  • colorize.jar (40Kb)
Adds syntax-coloring to the "Source" Tab. Requires JDK, not for JRE.
  • metrics.jar (coming soon)
Adds a "Metrics" tab to summarize diagrams and guide design.
  • table.jar (coming soon)
Adds a "Table" tab to summarize and edit diagrams.
  • style.jar (coming soon)
Adds a "Style" tab to set colors, fonts, and sizes.
   
If you download any of these, add them to your CLASSPATH ahead of argouml.jar.
