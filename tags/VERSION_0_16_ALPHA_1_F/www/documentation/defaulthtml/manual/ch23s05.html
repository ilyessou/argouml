23.5. Built In Stereotypes

UML 1.3 defines a large number of stereotypes which are (mostly) supported by ArgoUML.

Caution

Not all stereotypes defined by UML 1.3 appear in ArgoUML, due to limitations in the current implementation of some artifacts. The table below lists all stereotypes defined in UML 1.3 and if they are supported in ArgoUML or not.
