Preview of Upcoming Argo/UML Features in Argo/UML 0.7

We are working on many new features for the next release of Argo/UML.


Features coming in Argo/UML v0.7.x:

  • Java 1.2 support
  • New diagram types: Deployment
  • Improved navigational perspectives
  • Improved find utility: more search criteria
  • Design process guidance
  • More wizards
  • Selection-action buttons
  • Statistical usage monitoring
  • Better table views
  • Customizable code generation
  • XMI file format revisions
  • The return of the design history tab
