  Step Two: System Requirements
 


You will need the to have the following before you can use Argo/UML:

  • A Java 1.1.x or Java 1.2 Virtual Machine such as one of these:
    • Sun's JDK 1.1.7 or Sun's JDK 1.2
    • JRE 1.1.7 or
    • Blackdown's JDK

  • If you are using JDK 1.1.x, you need the Swing GUI Library
    • Sun's Swing 1.1.1 Unix Tar format
    • Sun's Swing 1.1.1 Windows Zip format
  • The XML4J XML 2.0.4 Parser Library (you must use this version, not the latest version)
    • XML4J.zip (you can get source for this library from IBM's alphaworks)


Argo/UML runs on any platform with JDK 1.1.x or Java 1.2. We have tested the following environments:

  • Sun Solaris 5.5.1 with Sun JDK 1.1.5 and JDK 1.1.6
  • Windows NT and 98 with Sun JDK 1.1.7
  • Windows NT and 98 with Symantec Cafe 2.5 and 3.0

For best results on Windows/NT you will need a 200+ Mhz processor, 64 Mb of RAM, and 10 Mb free disk space.
