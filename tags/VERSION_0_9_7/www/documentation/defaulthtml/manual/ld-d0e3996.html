The ArgoUML navigation pane perspectives dialog. A rectangular pane with two tabs labeled Perspectives (currently selected) and Panes. The top half contains a text area with a list of all the currently defined perspectives and to the right three buttons stacked vertically, labeled New, Remove and Duplicate. The second half contains two text areas. The one on the left is labeled Rules Library and contains a list of possible perspective rules. The one on the right is labeled Selected Rules and contains the rule list for the currently selected perspective (empty if no perspective is selected). Between these two areas are two buttons stacked vertically labeled >> and <<. At the very bottom right is a button labeled OK.
