Dialog box for New To Do Item…. At the top is a text box where a one line headline can be entered. Beneath that is a drop-down selector where the priority of the to-do item can be selected (High (default), Medium or Low). Beneath this is a text box for a URL where further information can be found. The main body of the dialog is then a text area for the detailed description of the to-do item. Finally at the bottom are two buttons, labeled from left to right Add and Cancel.
