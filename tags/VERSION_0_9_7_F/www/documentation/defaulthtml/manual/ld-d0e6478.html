The constraint editor. Along the top from the left are icons for creating a new constraint (grayed out), deleting a constraint (grayed out), exiting the editor without saving changes, validating a constraint and altering the OCL parameters. To the right of these is a tick box for the Syntax Assistant. Beneath this is a row of six drop down menus, labeled from left to right General, Basic Operators, Numbers, Strings, Booleans and Collections. The remainder of the screen is a text area labeled Edit constraint.
