ArgoUML Style Tab

Argo's Style Tab allows you to edit the visual properties of the selected design element. For example, you can set the background color, the text color, the font, and whether certain compartments or details are shown. The contents of this tab pane vary depending on the selected design element. Style panes have been defined for:

  • Diagram: background, grid, grid snap
  • Graphical Shapes: fonts, colors, size
  • Class: fonts, colors, stereotype, compartments
  • Association: fonts, colors, stereotype
  • State: fonts, colors, stereotype, compartments
  • Use Case: fonts, colors, stereotype, compartments

Note: In order to see the Style tab, you must download style.jar and add it to your CLASSPATH before starting ArgoUML.
