13.4. Flat/Hierarchical Button

To the right of the presentation selections is a button which toggles whether hierarchies are shown with “twisties” ( ) to indicate the hierarchy.

  • Hierarchical. The current presentation uses “twisties” to show the hierarchy. Button 1 click will on the icon will switch to a representation without them.

  • Flat. The current presentation does not use “twisties” to show the hierarchy. Button 1 click will on the icon will switch to a representation which has them.

Caution

The V0.14 version of ArgoUML (as described above) really has the flat/hierarchical icons logically back to front.
