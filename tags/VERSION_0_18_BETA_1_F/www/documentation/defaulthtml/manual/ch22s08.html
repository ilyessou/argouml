22.8. Interface

A component or deployment diagram may show components or component instances which implement interfaces. For details see Section 17.15, “Interface”.

Caution

The V0.14 release of ArgoUML uses the same representation of an interface as a class diagram. The UML standard suggests that an interface on a component or deployment diagram should just be shown as a small open circle, connected to the component which realizes that interface.

Warning

There is no way to show the linking of an interface to a component or component instance in the V0.14 release of ArgoUML.
