Known Incompatibilities

Here is a list of features which are not implemented yet and can be seen as a violation of the UML specification (1.3). These are organized in IssueZilla under Issue 1067 - Compliance to UML 1.3 Specification.

All Diagrams

  • Qualifiers are missing (Issue 499).
  • Optional directional triangle missing on association names (Issue 503).
  • List of provided multiplicities is not complete (Issue 1025).

Class Diagram

  • Association classes are missing. Though ArgoUML can read association classes from XMI files, it cannot display them correctly. That is due to a flaw in GEF, which needs a major rework.
  • Parameterized classes are missing
  • Packages can have stereotypes, but they are not displayed
  • Notes to any model element not implemented
  • Cannot add a method to an operation (Issue 954).
  • Cannot show collaborations on a class diagram (Issue 1067).

Object Diagrams

  • Currently not implemented, use a Deploymentdiagram
  • Attributes cannot have attribute values

Activity Diagrams

  • Swimlanes missing

Useful Optional Features

Class Diagram

  • Cannot specify package visibility as defined by Java (Issue 928).
