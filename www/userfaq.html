  Where can I obtain general information about ArgoUML?
      The best idea is to subscribe to the user mailing list at users@argouml.tigris.org. In order to subscribe please send an email to users-subscribe@argouml.tigris.org, with any mail body text you wish.

  How do I start the ArgoUML program?
      Just execute the following steps:
  • First, you need Java2 installed, get it from SUN for your platform.
  • Then you need to download a binary package of ArgoUML (ArgoUML-0.x.x.zip or ArgoUML-0.x.x.tar.gz). If you have this, unpack this archive (on Windows use WinZIP.
  • Now you should have a bunch of .jar files (argouml.jar, nsuml.jar and others...) in your directory, please do NOT unpack these. Just leave them as jar files.
  • At last you can just start the argouml.jar, on Windows platforms by double-clicking it, on other platforms by typing
    java -jar argouml.jar
    on a command line in the appropriate directory. Note that you need all jar files in this directory. That's it, ArgoUML should start up now. If not, please consider asking on the users mailing list.

  How to create activity/state diagrams, they are greyed out?
      In order to create activity diagrams, a class or a use case has to be selected.
In order to create state diagrams, a class has to be selected.
       
  How to draw self associations?
      Just execute the following steps:
  • Click on the association-icon in the Class Diagram Toolbar (not the rapid buttons).
  • Press left mouse button on the Class, continue holding down button for the next step.
  • While holding down left mouse button, move the mouse cursor about an inch away from the class (a rubber band line will follow the mouse cursor at this point).
  • Lift mouse button.
  • Click on corners of Association until clicking back on Class.
  • Done.
The same principle works in all other diagrams, e.g. with self transitions to states in state diagrams.
