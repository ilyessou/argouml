  New features in ArgoUML v0.5.2:
  • Activity Digrams
  • Table views
  • Better arc editing
  • Find and "GoToDiagram" Dialogs
  • More critics, more efficent
  • New "Broom" alignment tool
  • Clarifier icons on arcs
  • Multi-page printing
  • Double-click on "to do" item jumps to offending design elements
  • Adjustable grid snap
