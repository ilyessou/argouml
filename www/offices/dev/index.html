Argo/UML Virtual Offices: Development
Mailing List: argo-dev@egroups.com

Development includes all the Java code that implements Argo/UML, and supporting code such as shared libraries and utilities. This is really the heart of the project, and the most active area. Here we discuss new features, track bugs, and help each other learn how the existing code works.

Goals
  Implement Argo/UML infrastructure and features
  Comply with applicable standards
  Lay foundations for later development
  Release new versions
  Fix bugs as they arise
