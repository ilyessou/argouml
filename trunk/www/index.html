Download: Two Ways to Launch ArgoUML

Launch via JavaWebStart:
1. Install JavaWebStart.
2. Launch latest stable release

Installation and upgrades are automatic. If you have problems, please see our JWS FAQ.

Download files:
1. Download latest stable release: v0.12 (other formats)
2. Unzip the archive
3a. On Windows or Mac: Double-click on argouml.jar
3b. On Unix: Type the command 'java -jar argouml.jar'

Screenshot (click for tour)

ArgoUML's user interface is divided into 4 panels:
Top left: a hierarchical view of the current project file.
Top right: editor for the selected part of the project, in this case a class diagram.
Bottom left: the designer's "to do" list.
Bottom right: details of the selected object in the diagram or the selected "to do" item.

Supported diagrams:

  • Class
  • State
  • Use case
  • Activity
  • Collaboration
  • Deployment
  • Sequence

For Developers of ArgoUML

ArgoUML is not only a free UML modelling tool, it is also an Open Source Development project where you are invited to contribute.

Go to the Developer Zone to read about the project and to find the documentation of the internals of ArgoUML.

As of October 2002 developers of ArgoUML can use RefactorIT, a java based tool to assist in refactoring and code analysis. Just register, download and request the opensource license, which allows you to work with a number of Open Source projects and up to 50 of your own classes, which would even allow you to work on smaller modules outside the ArgoUML namespace.

Thanks to RefactorIT for their support.
