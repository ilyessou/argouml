ArgoUML homepage

Welcome to the homepage of ArgoUML, a java based cognitive CASE tool.

ArgoUML is a modelling tool to help you do your design using UML. A commercial extension is available from Gentleware.

Look at the features that are already implemented in ArgoUML!

For developers of ArgoUML

ArgoUML is not only a free UML modelling tool, it is also an Open Source Development project where you are invited to contribute.

Goals

The goal of the ArgoUML project is to build an object oriented design tool that is:

  • a joy to use (we really mean it)
  • actually helpful to designers when they are making design decisions, by offering cognitive support through critics
  • completely open source Java (FreeBSD license)
  • supporting UML version 1.3
  • Modular and extensible,
  • integrated with the web and other Tigris tools.
