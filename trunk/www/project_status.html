ArgoUML project status

Roadmap

Currently we are working towards release 0.9.6, which will be an interim release before 0.10. The 0.9.x branch of ArgoUML was derived from the 0.8.x series after which a number of changes were introduced.

A plan when we want to reach release 0.9.6 is here.
