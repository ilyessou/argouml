Argo/UML project status

Roadmap

Currently we are working towards release 0.9.6, which will be an interim release before 0.10. The 0.9.x branch of Argo/UML was derived from the 0.8.x series after which a number of changes were introduced.

A plan when we want to reach release 0.9.6 is here.

Recent activities

This is an incomplete list of recent activities

  • plugin module which allows us to write plugins which are independent from the main code and functionality.
  • UML Notation in diagrams
  • improved user documentation and website
  • improved handling of Issuezilla, our bug database
  • lots of bug fixes
  • Load and save issues

This list is incomplete, if you want to get a better feeling for the recent activities you are welcome to join the developer mailing list.
