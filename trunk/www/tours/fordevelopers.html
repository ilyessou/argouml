16dec01 Alicebot install : Ever dream of having a real live tutor hovering over your shoulder to guide you? The cognitive creature in Argo is hardcoded but considerable advances in AI have enabled looser bindings between data and logic...and has given birth to a whole new set of service and help tools. Unleashing the power of AI using AIML of DAML in java applications is getting closer and closer. This tutorial will show you how to install Alicebot. Imagine, combining the power of the UML interface with an AI device...you could peer into the brains of the next wave of super smart applications!

Running with the new ant : ant knocks a DOS tree down! This is a surprising problem, though it is a known bug...Ant cannot see the ANT_HOME in the autoexec.bat! The new ant 1.4.1 from Apache. The fix is here:

path C:\jdk1.3.1\bin;C:\jakarta-ant-1.4.1\bin
set JAVA_HOME=C:\jdk1.3.1
set ANT_HOME=C:\jakarta-ant-1.4.1
