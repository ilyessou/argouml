Notice in the Explorer (top-left) that the attributes of the Course class appear.
Below the Course class, add a new class called 'Student' as is done in Step#1.
Change the color of the Student class, using the Style tab located in the Details pane (bottom-right). Change the fill color to yellow and the border to red. Add 2 new attibutes, calling them newAttr and studentID. Add a new Operation and call it takeTest. The operation parameters have to be added by right-clicking in the parameter list on the operation's property panel. Once you have added the parameter you can call it 't'. You need to set the parameter type to 'Test'. To do this you have to create a new class called 'Test', then it will be available in the parameter type list on the parameter property panel.
