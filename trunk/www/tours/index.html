Interactive demonstrations

The technique used is called Viewlets and your ordinary JavaApplet-enabled browser will be enough to view it.

Please note, the Viewlets themselves are residing on web server, and sometimes that server goes down. In that case, try again later!

  • Robocode State Diagram under ArgoUML will show you what a state diagram looks like under Argo and gives some ideas of the uses of State diagrams. The demonstration zargo project was contributed by Erik Dyrelius. The code is based on Robocode. Worth a look!

  • Lake Cruise with ArgoUML A somewhat random glide through Argo. See below.

  • Initial tour will guide you through starting ArgoUML from the command line, reverse engineering entire Java source packages, a brief overview of the documentation features, and a very quick introduction to state diagrams.

There are some more Viewlets targeted for the Developers of ArgoUML.
