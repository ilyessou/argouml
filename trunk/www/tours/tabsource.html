ArgoUML Source Tab

ArgoUML's Source Tab allows you to preview the source code that will be generated for the selected design element. The target language can be selected from a combo box. A second combo box allows selection of the file, for the case where more then one file is generated for the selected modelelement, such as e.g. with C++.

A standard installation of ArgoUML includes only Java. With the aid of plug-ins, also the languages C++, PHP and C# are available. Other languages may become available with future versions of ArgoUML.
