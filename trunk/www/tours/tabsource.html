ArgoUML Source Tab

ArgoUML's Source Tab allows you to preview the source code that will be generated for the selected design element. The target language can be selected from a combo box. A standard installation of ArgoUML includes UML and Java. With the aid of plug-ins, also the languages C++, PHP and C# are available. Other languages may become available with future versions of ArgoUML.

In future versions, ArgoUML will allow you to edit code directly in this tab pane and update the UML model as a result.
