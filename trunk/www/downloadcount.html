The headers above mean:

  • zip - zip downloads.
  • tar.gz - tar downloads from argouml-downloads page.
  • appb - appbund downloads from argouml-downloads page.
  • jws - java web start starts.
  • ls - java web start starts of latest stable.
  • Manpdf - Number of pdf downloads of the manual.
  • Manidx - Number of views of the first page of the user manual (from the argouml-stats project).

How do we count?

The counts are based on Unique Views from Google Analytics. We count the click of links to .zip or .tar.gz files of the release using an onClick javascript command. If the browser doesn't do this, the download or start is not counted. The way Google Analytics work we also count failed attempts to do downloads. There is not much we can do about it. For Java Web Start we count the amount of Unique Views of the jnlp file.
