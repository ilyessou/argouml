How do we count?

These counts are based on the download counts from our web site.

In an attempt to let the figures reflect the amount of people that have installed ArgoUML every downloaded .zip or .tar.gz archive of the release is counted as one. This could be too much since sometimes people download without installing. This could be too little since sometimes people download once and then install on many machines.

For Java Web Start we count the amount of times the last file in the Java Web Start set it downloaded. This is the best we can do since after that is downloaded, the Java Web Start installation is completed and ArgoUML starts.
