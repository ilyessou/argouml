11.5. The Drawing Grid

The editing pane is provided with a background grid which can be set in various styles or turned off altogether through the menu (see Section 9.5.5, “Adjust Grid”).

Whatever grid is actually displayed, placement of items on the diagram is always controlled by the setting for grid snap, which ranges from 4 to 32 pixels (see Section 9.5.6, “Adjust Grid Snap”).
