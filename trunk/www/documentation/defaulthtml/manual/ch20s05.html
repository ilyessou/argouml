20.5. Call Event

This artifact is described in the context of state diagrams (see Section 18.6).

Caution

The only place this is used is for the trigger of a transition. However remember that action states do not have explicit triggers. The transition is implicitly triggered as soon as the entry event of the action state is complete. An explicit trigger should not therefore be set.

The current release of ArgoUML will not check that this constraint is met.
