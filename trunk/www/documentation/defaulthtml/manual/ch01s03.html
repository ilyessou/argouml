1.3. Overview of the User Manual

1.3.1. Tutorial Manual Structure

Chapter 2 provides an overview of UML based OOA&D, including a guide to getting ArgoUML up and running.

Chapter 3 through Chapter 6 then step through each part of the design process from initial requirements capture through to final project build and deployment.

As each UML concept is encountered, its use is explained. Its use within ArgoUML is then described. Finally the case study is used to give examples of the concept in use.

1.3.2. Reference Manual Structure

Chapter 7 is an overview of the user interface. Separate chapters Chapter 8 through Chapter 12 describe each of the sub-windows of the user interface, known as Panes, and the menu bar.

Chapter 14 is an overview of the artifacts with ArgoUML. The following chapters (Chapter 15 through Chapter 22) describe, the artifacts that can be created through each ArgoUML diagram, and their properties, as well as some standard artifacts provided with the system.

A complete Glossary is provided. Appendix A provides material to supplement the case study used throughout the document. Appendix B and Appendix C identify background information on UML and UML CASE tools. Appendix D describes the plan for creation of this manual. Appendix E gives some guidelines for those writing this manual. Appendix F is a copy of the GNU Free Documentation License.

A future ambition is to provide a comprehensive index
