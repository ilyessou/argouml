9. Logging

9.1. How do I ...?

  • ... create logging entries?

    ArgoUML uses the standard Log4j logging procedures. This is not fully implemented yet at the time of writing. However in org.argouml.resources you will some example log configuration files. Modify these to your needs.

    In order to invoke ArgoUML with logging support give it the command line parameter -Dlog4j.configuration=org/argouml/resource/your-file.lcf. You also need to make sure that the log configuration file is in the build directory, so do an ant package for example before starting ArgoUML.
