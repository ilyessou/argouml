10.1. Introduction

The documentation (currently manual, cookbook, and quickguide) is written using DocBook XML V4.1.2. This section covers some conventions for use of DocBook and for the documentation in general. It also includes some information for tooling configuration, e.g. for Emacs with the psgml package.
