2.1. System requirements

Minimum system requirements:

  • Any OS that supports Java2

  • 200MHz Intel Pentium, AMD K6 or similar processor

  • 64MB (128MB for Windows 2000) RAM

  • 10MB of free disk storage space on your hard disk

  • SVGA video output and monitor

  • Mouse (or other pointing device) and a Keyboard

  • Java 2 JRE or JDK.
