3.1. The window

ArgoUML contains one model that your can examine and modify.

Commands are available from the menus and by right-clicking on the objects either in the list of objects or in the diagram. The commands available from the objects work on these objects.

Some commands for creating and drawing are available from the icon bars over the diagrams.

3.1. The window

At the top there is a menu bar with commands available. In the file menu you can store the model or open another model instead.

The upper left part of ArgoUML lists the diagrams and objects of the model in one of several views.

The upper right part of ArgoUML shows one diagram at the time. You can work with the objects in the diagrams, dragging and dropping and using the quick-links to create new objects connected to the already present objects.

The lower right part you contains various details of various objects of the model. You select the object in one of the upper levels and choose what details you want to examine using the tabs.

The lower left part contains a list of all ToDo items for this model.
