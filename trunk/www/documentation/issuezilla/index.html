ArgoUML Issuezilla components

There are two components in ArgoUML: ArgoUML and website.

The ArgoUML component contains all errors in the argouml product and all problems are connected to a release of ArgoUML. There are subcomponents for each part of the code.

The website component contains errors on the web site. It is things that are not connected to any release of ArgoUML and problems that we have identified that is actually problems or wishes with the Tigris infrastructure. There are subcomponents for different areas.
