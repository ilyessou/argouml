UML Books and Tools


Books about UML:

The Unified Modeling Language User Guide

The must-have guide to UML. Well organized, complete, and clear. Many "study guide" elements. Advertised as a tutorial, but more like a well explained introductory reference with small examples.

 

UML Distilled

This is a quick and simple introduction to UML. It clearly describes all the UML diagram types and gives advice based on the author's broad experience in system development. I started with this book myself.

         

Objects, Components, and Frameworks With Uml :
The Catalysis Approach

 

Unified Modeling Language Reference Manual

This is the authorative reference manual on UML, a must-have for anyone serious about advanced usage of UML.

         

Use Cases Combined with
Booch, OMT, UML

Detailed description of a development method. Dozens of forms, time tables, and spreadsheets to use in managing your next development project.

 

UML in a Nutshell : A Desktop Quick Reference

         

Understanding UML

 

The Object Constraint Language

The Object Constraint Language (OCL) is a very useful advanced feature of UML that can add more meaning to your UML designs. OCL will become a central part of ArgoUML.

         

Objects, Compontens, and Frameworks with UML

 

UML Toolkit

A good introduction to UML. Includes a Rational Rose CD. This book has successfully been used in undergraduate classes.
