ArgoUML User's FAQ

  • How to create activity/state diagrams, they are greyed out?
  • How to draw self-associations?




  How to create activity/state diagrams, they are greyed out?
      In order to create activity diagrams, a class or a use case has to be selected.
In order to create state diagrams, a class has to be selected.
       
  How to draw self associations?
      Just execute the following steps:
  • Click on the association-icon in the Class Diagram Toolbar (not the rapid buttons).
  • Press left mouse button on the Class, continue holding down button for the next step.
  • While holding down left mouse button, move the mouse cursor about an inch away from the class (a rubber band line will follow the mouse cursor at this point).
  • Lift mouse button.
  • Click on corners of Association until clicking back on Class.
  • Done.
The same principle works in all other diagrams, e.g. with self transitions to states in state diagrams.
