Proposed structure of ArgoUML including non-CVS directories.

Directories are prefixed with a slash (/), files are prefixed with a minus (-).
Files and directories followed by an asterisk are not included in CVS.
Files and directories followed by a pound sign (#) would only be contained in CVS as branches, and would no longer be part of the CVS base.

Restructuring of the source would improve the compartmentalization of ArgoUML. Breaking the source into smaller components should allow simpler maintenance ant quality control.

Adding an additional directory level under the sources would facilitate splitting the source files into smaller components in the same manner as NetBeans or Eclipse.
