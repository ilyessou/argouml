5.6. Help System

Purpose - to provide the menu actions that starts the help and other documentation. To provide infrastructure that makes context sensitive help possible.

The Help System is not yet implemented.

The Help System will be located in org.argouml.help.

The Help System is a Layer 1 component. See Section 4.4.

Javahelp or some other help function will probably be used.
