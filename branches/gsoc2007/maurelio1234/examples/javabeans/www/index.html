Welcome to the new Sequence Diagram on ArgoUML project

Mission

The mission of this project is to provide a new implementation to the Sequence Diagram subsystem on ArgoUML.

The Sequence Diagrams are going to be reimplemented on ArgoUML under the Google Summer of Code program.

How can I collaborate?

  • Download the source and test it.
  • Use the issue tracker to track features and enhancement requests. It is not just for bugs, pre-populate it with feature descriptions too.
