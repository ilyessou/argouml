ArgoUML: A modelling tool for design using UML.
Welcome to the homepage of ArgoUML, a java based cognitive CASE tool.

ArgoUML is a modelling tool to help you do your design using UML.

Look at the features that are already implemented in ArgoUML!

For Developers of ArgoUML

ArgoUML is not only a free UML modelling tool, it is also an Open Source Development project where you are invited to contribute.

Go to the Developer Zone to read about the project and to find the documentation of the internals of ArgoUML.

As of October 2002 developers of ArgoUML can use RefactorIT, a java based tool to assist in refactoring and code analysis. Just register, download and request the opensource license, which allows you to work with a number of Open Source projects and up to 50 of your own classes, which would even allow you to work on smaller modules outside the ArgoUML namespace.

Thanks to RefactorIT for their support.
