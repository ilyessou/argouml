  Step Two: System Requirements
 


You will need the to have the following before you can use ArgoUML:

  • A Java 1.2 Virtual Machine such as one of these:
    • Sun's JDK 1.2
    • JRE 1.2


ArgoUML runs on any platform with JDK 1.2. We have tested the following environments:

  • Sun Solaris 5.5.1 with Sun JDK 1.2
  • Windows NT and 98 with Sun JDK 1.2
  • Linux with Sun JDK 1.2

For best results on Windows/NT you will need a 300+ Mhz processor, 64 Mb of RAM, and 10 Mb free disk space.
