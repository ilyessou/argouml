9.1 Class Diagrams: Contribute!

9.1.1 Class Diagrams

Is it possible to switch to 'real' UML Syntax, like 'Name : String' in class diagrams?

No, but it is planned to make that the default.

I see there is a note icon in the tool bar. How do I use it?

Click on your target class. Then click on the note icon. Argo will generate the link automatically. The notes will be implemented for many UML artifacts in the next release .9.6

You can also right click to add a note as well! Be aware that you can add an undefined number of notes to any one class!

Be aware that your note will not appear in the source code documentation tab. But it will appear in the diagram view of the navigation pane (the left pane tree view).

Please see to do listhttp://argouml.tigris.org/jobjar.html.

What is the easiest way to enter data into attributes and methods window for classes?

Click direct class properties pane and start typing.

In fact, it would be interesting to see if you can type 'stereotypes' write in the class attribute box for generating XML diagrams.
