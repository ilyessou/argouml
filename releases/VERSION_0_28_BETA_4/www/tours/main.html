ArgoUML Main Window

ArgoUML's main window has a toolbar, menu bar and four main panes: explorer, editing, to do, and details.
The panes can be resized by dragging on the divider bars between them.

Click on one of the items below for more information about the tabs in the details pane:

  • ToDo Item Tab
  • Properties Tab
  • Documentation Tab
  • Presentation Tab
  • Source Tab
  • Constraints Tab
  • Tagged Values Tab
