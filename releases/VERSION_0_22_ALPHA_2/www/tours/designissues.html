ArgoUML Design Issues Window

ArgoUML's Design Issues Window allows you to specify which design decisions you are working on. The window shows a list of the types of decisions that object-oriented software designers must make. Each decision has a priority from zero (off) to five (high priority). Each critic supports one or more types of decisions. So, if all supported decisions have priority zero, then that critic will be deactivated.
