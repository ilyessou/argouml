How to Add a Recipe

Thanks for contributing to the development of Argo/UML. If you know how to accomplish a coding task with Argo/UML and you think others would benefit from your knowledge, please add a recipie to our web site. Here's how.

Ingredients

Files and tools you will need:

  • This web page template
  • An HTML editor: such as HomePage, VisualPage, or a simple text editor

Directions

  • Use your web browser to save this page to your local system.
  • Use an HTML editor to replace these instructions with your new instructions.
    • Add a new title and recipe description
    • List ingredients including files, images, and reference documents
    • Give step-by-step instructions
    • Summarize the expected results
    • Include your name and email address
    • Use images as needed
  • Place your new recipe on your web site and email the URL to Jason Robbins , or simply email the entire page.
  • After we review your submission, we may place it on the Argo/UML web site.

Results

A new recipe that other contributors can use to enhance Argo/UML.

Contact

Please send comments and questions to: Jason Robbins
