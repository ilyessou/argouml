  Step Five: Download Example Files
 


The following optional files contain examples of designs done with ArgoUML:

  • enroll.zip (10 kb)
    This example contains one class diagram, two state diagrams, one use case diagram, and one activity digram. The system being modeled is a university enrollment system.
  • tinygef.zip (8 kb)
    This example contains several class diagrams that document     GEF (the graph editing framework) which is part of ArgoUML's own implementation. This is just an example to show what ArgoUML can do, it is not intended as complete documentation on GEF.
