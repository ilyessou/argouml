Step Five: Browse Example Files

  • enroll.zip (22 Kb) download and unzip this file to get enroll.argo.     Requires Argo/UML v0.5.2.
    This example contains one class diagram, two state diagrams, one use case diagram, and one activity digram. The system being modeled is a university telephone enrollment system. Basically, students call the system to enroll in classes. This is just an example to show what Argo/UML can do, it is nowhere near complete.
  • tinygef.zip (22 Kb) download and unzip this file to get tinygef.argo.     Requires Argo/UML v0.5.2.
    This example contains several class diagrams that document     GEF (the graph editing framework) which is part of Argo's own implementation. This is just an example to show what Argo/UML can do, it is nowhere near complete.
