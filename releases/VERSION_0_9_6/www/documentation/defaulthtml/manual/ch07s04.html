7.4. General Information About Panes

The four sub-windows of the main ArgoUML window are called panes. Clockwise from top left these are the navigator pane (see Chapter 8. The Navigation Pane), editing pane (see Chapter 9. The Editing Pane), details pane (see Chapter 10. The Details Pane) and to-do pane (see Chapter 11. The To-Do Pane). At the top the editing pane is a tool bar.

7.4.1. Re-sizing Panes

You can re-size panes by dragging on the divider bars between them. The mouse cursor changes to a vertical or horizontal bar with an arrow at right-angles when over the divider bars.

7.4.2. Looking at Different Diagrams Simultaneously

ArgoUML provides the ability to spawn copies of key windows. This is possible for the editing pane and for the tabs of the details pane. These can then be manipulated as separate windows and placed as desired on the screen.

This is achieved through use of button 1 double click as described in sections Section 9.2 and Section 10.2.

Note

There is no equivalent function for the navigation or to-do panes.

7.4.3. Reducing the Number of Panes Visible

This function is not implemented directly. However you can spawn a pane (see Section 7.4.2) and maximize this to be the whole window, which achieves the same thing.
