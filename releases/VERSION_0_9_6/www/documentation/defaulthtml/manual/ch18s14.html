18.14. Shallow History

Shallow History is a pseudostate (see Section 18.8) within a composite state, which causes control to be passed to the state that last had control within the composite state. A transition may show a default to state to use if the composite state has never been entered before.

The shallow history ignores any composite states within this composite states, dealing only with direct sub-states.

A shallow history is represented on the diagram as a circle containing the letter H.
