1.2. About the project

ArgoUML is an open source project, so it depends on people that volunteer to work on it. Especially in the area of development there is still so much to do! Everyone who wants to contribute needs to know the what, where and how to write good quality code, but getting into it is a big challenge since there are hundreds of Java files. This developer Cookbook is dedicated to all interested programmers and should help to transfer the knowledge from the old experts to them. Please feel free to send more questions and/or answers to the dev mailing list!
