1.1 The aims of ArgoUML

The goal of the ArgoUML project is to build an object oriented design tool that is:

  • A joy to use (we really mean it),
  • Actually helpful to designers when they are making design decisions,
  • Completely open source Java,
  • Leading edge (supports UML 1.3)
  • Modular and extensible,
  • Integrated with the web and other Tigris tools.
For a more comprehensive list of features, look at the ArgoUML features page.

The ArgoUML website itself, strives to give you a resource for everything which has something to do with the Unified Modeling Language. We therefor welcome your contributions. Whether you know a book, have data about a conference, useful UML links to sites, mailinglists or user groups, drop us a line at [mailto://umlresources@argouml.tigris.org] and we will include it in our UML resources section.
